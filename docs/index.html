1. Project Overzicht

De ActiZ Document Vergelijker is een AI-gedreven tool die automatisch verschillen detecteert tussen documentversies en deze analyseert op betekenis en impact. Specifiek ontwikkeld voor de zorgsector om beleidswijzigingen snel en accuraat te kunnen beoordelen.

🎯 Hoofdfunctie

Vergelijkt PDF documenten en genereert AI-analyses van wijzigingen met focus op beleid, financiën en stakeholder impact.

🔧 Technologie

Python + Flask web app met Azure Document Intelligence en Azure OpenAI (GPT-4) voor AI-analyse.

👥 Doelgroep

ActiZ medewerkers die beleidswijzigingen moeten analyseren en adviseren over impact op zorgorganisaties.

✅ Huidige Status (Production Ready)

Core Functionaliteit

  • ✅ PDF upload en tekstextractie
  • ✅ Document vergelijking met inline diff

Probleem: Flask debug mode threading errors op Windows

Oplossing: Gebruik debug=False in productie

app.run(debug=False, port=5000, host='127.0.0.1')

Oplossing: Check template path en Bootstrap CDN loading
